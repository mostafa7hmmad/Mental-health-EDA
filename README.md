# Mental-health-EDA
![Mental](mental.jpg)
This project delves into the profound impact of mental health conditions on individuals' lives. By analyzing a dataset that includes factors like work hours, physical activity, education level, stress levels, hours of sleep, and health history, the project aims to uncover patterns and correlations.
